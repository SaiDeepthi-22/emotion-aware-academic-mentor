An intelligent **Streamlit-based academic mentor** that adapts explanations based on a student's **emotional state** such as confusion, stress, confidence, or normal understanding.

##  How It Works

The app analyzes the student's input and detects emotion using keyword-based logic:

- **Confused** ‚Üí Step-by-step slow explanation  
- **Stressed** ‚Üí Calm, supportive explanation  
- **Confident** ‚Üí Fast explanation with advanced insights  
- **Normal** ‚Üí Standard clear explanation  

Then it uses a **Transformer model (`google/flan-t5-base`)** to generate responses accordingly.
